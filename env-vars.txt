AEON-V1 ENVIRONMENT VARIABLES
==============================
Update this file whenever a new variable is added.

--- LLM / AI ---

AEON_V1_LLM
  Enable LLM reasoning (Layer 4). Set to 1 to enable, 0 to disable.
  Default: 0

.env.lmstudio.template
  Generic LM Studio template file for local .env setup. It contains placeholder
  model IDs so any LM Studio model can be swapped in without machine-specific
  values.

AEON_V1_LLM_MODEL
  Local model name passed to LM Studio.
  Examples: google/gemma-4-e4b, qwen/qwen3-4b-2507
  Default: local-model

AEON_V1_LLM_CHAT_MODEL
  Model name used for interactive chat responses. Use the fastest useful local instruct model.
  Default: AEON_V1_LLM_MODEL

AEON_V1_LLM_DEEP_MODEL
  Model name reserved for deeper background reasoning work.
  Default: AEON_V1_LLM_MODEL

AEON_V1_LLM_MUSIC_MODEL
  Model name used only for explicit song-library management requests.
  Default: AEON_V1_LLM_CHAT_MODEL

AEON_V1_LLM_BASE_URL
  Base URL for the LM Studio OpenAI-compatible local server.
  Default: http://localhost:1234/v1

AEON_V1_LLM_MUSIC_BASE_URL
  OpenAI-compatible endpoint for the machine-local music worker. This may point
  to another tailnet machine while Aeon's identity and memory remain centralized.
  Default: AEON_V1_LLM_BASE_URL

AEON_V1_LLM_MUSIC_TIMEOUT
  Request timeout in seconds for the distributed music role.
  Default: 90

AEON_V1_MESH_HUB_URL
  Base URL of the Agent Mesh hub (on T5810) used to dispatch accepted music
  proposals as audited, human-gated approvals. Empty => manage_music only
  prepares the task and never leaves the machine.
  Default: (empty)

AEON_V1_MESH_TOKEN
  Bearer token for the Agent Mesh hub API (AGENT_MESH_TOKEN on the hub).
  Default: (empty)

AEON_V1_MESH_MUSIC_AGENT
  Mesh agent id of the T3610 music station that claims and runs the task.
  Default: music@t3610

AEON_V1_MESH_MUSIC_CWD
  Working directory the station should use for the music task.
  Default: /mnt/jellymedia1

AEON_V1_MESH_TASK_TTL
  Seconds the pending approval stays valid before it expires (hub allows 30-900).
  Default: 900

AEON_V1_MESH_TIMEOUT
  Request timeout in seconds for Agent Mesh hub calls.
  Default: 15

AEON_V1_LLM_TIMEOUT
  Request timeout in seconds. Increase for slower local models.
  Default: 120

AEON_V1_LLM_MAX_TOKENS
  Maximum tokens the LLM will generate per call. Lower this for faster responses on small local models.
  Default: 1200

AEON_V1_LLM_REASONING_EFFORT
  Reasoning effort passed to LM Studio for local reasoning models. Use low for faster local chat.
  Default: low

AEON_V1_LLM_TOOL_CALLING
  Enable tool-calling path for reflect/simulate. When True, the LLM queries the MemoryIndexAgent
  instead of receiving all memories inlined in the prompt.
  Default: 0

AEON_V1_LLM_MAX_ATTEMPTS
  Max retry attempts for lmstudio calls before giving up.
  Also caps the number of tool-call rounds in generate_with_memory().
  Default: 1
